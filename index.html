Glyphicons for Bootstrap

The official Bootstrap icon font, featuring 290 glyphs from Glyphicons, ready for use in any Bootstrap project. Includes support for IE8+.

  • GitHub project
  • Bootstrap
  • Bootstrap Expo
  • Version 3.3.5

How to use

  1. Include the fonts

    Copy over all the font files to a /fonts directory near your CSS.

  2. Include the CSS or SCSS

    You have two options for enabling the fonts in your project: vanilla CSS or source SCSS. For vanilla CSS, just include the compiled CSS file from the /css in the repository.

    For SCSS, copy over the .scss files from /scss into your existing Bootstrap directory. Then import it. Recompile when ready.

  3. Add some icons!

    After you've added the fonts and CSS, you can get to using the icons. For example, <span class="glyphicon glyphicon-ok"></span>.

Bootstrap is a front-end framework for fast, sleek, and mobile-first web development.
